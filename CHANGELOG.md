# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-03-21

### Added
- MapLibre GL JS 5 + Deck.gl 9 spatial engine
- Si & Midorikawa (1999) GMPE intensity computation via Web Workers
- Real-time USGS + JMA earthquake monitoring via Durable Objects (10s poll)
- PLATEAU 3D building tiles for 35 Japanese cities
- AIS vessel tracking via AISstream.io
- Rail network status via ODPT
- Infrastructure fragility assessment calibrated on 5 historical disasters
- Nankai Trough multi-segment scenario engine with SharedArrayBuffer
- 13 infrastructure asset classes with exposure scoring
- Operator-grade dark UI with frosted glass panels
- Command palette (Cmd+K) and full keyboard navigation
- Depth cross-section panel (auto-triggers at 45° pitch)
- PWA support with stale-while-revalidate cache
- i18n support (Japanese, English, Korean)
- Cloudflare Workers API with Hono
- Neon Postgres database with PostGIS spatial queries
- R2 CDN feed architecture for zero-latency snapshots
